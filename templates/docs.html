About the API

As of now, our API provides 2 official endpoints:

- get_image
- api/img

The API is written in a micro python-framework, Flask.
GET_IMAGE endpoint is pretty much useless on its own unless used with the second API endpoint.
API/IMG endpoint will retrieve a response formatted JSON-like.
It will contain a direct URL to the image and the captcha solution.
